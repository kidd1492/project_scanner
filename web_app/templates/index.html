Project scanner Information

Introduction

The Project Scanner analyzes an entire codebase and converts it into structured, easy‑to‑understand data. It scans the filesystem, identifies routes, functions, classes, triggers, and internal call relationships, and then presents everything through a multi‑page dashboard. The system helps developers understand architecture, visualize execution flows, generate documentation, and uncover potential risks.

Traces

The Traces system reconstructs complete end‑to‑end execution chains across the application—from an HTML trigger, to the JavaScript function it calls, to the API route it hits, to the service method behind that route, and finally to the core Python function performing the work. This gives developers a clear picture of how user actions flow through the system, making debugging and architectural reasoning dramatically easier.

Mermaid Diagrams

The Mermaid Diagrams page converts trace chains into visual sequence diagrams using Mermaid syntax. Each UI‑to‑backend flow is rendered as a clear, readable diagram that helps developers understand system behavior at a glance. These diagrams can be viewed directly in the dashboard and copied for documentation, architecture reviews, or onboarding materials.

Template Builder

The Template Builder transforms your analysis data into ready‑made documentation and code templates. By loading all JSON reports—routes, JS functions, classes, functions, and traces—it can automatically generate API docs, Markdown summaries, sequence diagrams, and boilerplate code. This turns your analyzers into a full documentation engine that saves time and improves consistency.

Brainstorming

The Brainstorming page acts as an AI assistant that understands your project. It loads all analysis data and provides architecture suggestions, refactoring ideas, naming improvements, missing tests, error‑handling gaps, and potential abstractions. It’s essentially a context‑aware design partner that helps you improve the structure and quality of your codebase.

Security Scanner

The Security Scanner reviews HTML, JavaScript, and Python files for common vulnerabilities and risky patterns. It detects unsafe fetch calls, missing error handling, hardcoded URLs, missing input validation, dangerous imports, and potential SQL‑injection‑like patterns. It outputs a security.json report along with an AI‑generated summary and recommended fixes to help strengthen the codebase.
